Software

How do I access the applications already installed in Swift Linux?

There are several ways:
  • Click on the "Use Apps" icon on the desktop to see what software is accessible through the /usr/share/applications directory.
  • Main Menu -> Applications
  • Selected applications are also accessible through the clickable icons on the desktop.

How do I add software

Click on the "Get Apps" icon on the desktop to access the Software Wizard. (You can also access the Software Wizard through the Control Center.) The Software Wizard points you towards Synaptic and the Software Center.
