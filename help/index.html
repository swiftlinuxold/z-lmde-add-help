Welcome to Swift Linux!

Basics

  • Your user directory is /home/(your username) . Clicking on the Home icon in the upper left corner of the desktop opens up this directory.
  • The Control Center provides easy access to various configuration wizards, including lxrandr (configures the monitor display), the printer wizard, and the system configuration wizard.
